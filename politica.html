O responsável pelo aplicativo Ativa Trabalho respeita a tua privacidade e compromete-se a proteger os teus dados pessoais de acordo com a Lei n.º 41/VIII/2013, de 23 de dezembro (Regime Jurídico Geral de Proteção de Dados Pessoais das Pessoas Singulares), alterada pela Lei n.º 121/IX/2021, de 17 de março, e demais legislação aplicável em Cabo Verde.

Esta política explica como recolhemos, usamos, armazenamos e protegemos os teus dados.

O aplicativo Ativa Trabalho é uma plataforma móvel destinada exclusivamente a utilizadores em Cabo Verde, com foco inicial na ilha de Santiago e expansão gradual para outras ilhas. Ele conecta trabalhadores desempregados ou disponíveis com empreendedores (contratantes), facilitando oportunidades de emprego. Os trabalhadores não podem visualizar dados uns dos outros, e os empreendedores não podem visualizar dados de outros empreendedores. O acesso aos dados é restrito e requer criação de conta e login.

O aplicativo é gratuito, sem publicidade, e destinado apenas a maiores de 18 anos.

1. Âmbito de Aplicação

Esta Política aplica-se ao aplicativo móvel Ativa Trabalho (disponível para Android). Ela abrange todos os utilizadores: trabalhadores (candidatos a emprego) e empreendedores (contratantes), bem como administradores.

2. Dados Pessoais que Recolhemos

Para utilizar o aplicativo, é necessário criar uma conta e efetuar login. Recolhemos os seguintes dados pessoais, fornecidos diretamente por ti durante o registo ou atualização do perfil:

  • Nome completo
  • Email
  • Número de telefone
  • Data de nascimento
  • Nível de escolaridade
  • Aptidões profissionais
  • Ilha e concelho de residência
  • Informação aberta e relevante para contratação (ex.: experiência adicional ou disponibilidade)

Além disso:

  • Dados de login: Geridos pelo Firebase (serviço do Google), incluindo recuperação de senha via email (sem armazenamento de senhas em texto simples).

Não recolhemos dados sensíveis (ex.: origem racial, dados de saúde, orientação sexual) a menos que sejam voluntariamente incluídos na "informação aberta" e relevantes para contratação.

Os administradores têm acesso a ferramentas de gestão de utilizadores, mas o acesso é limitado ao necessário para manutenção do sistema.

3. Finalidades do Tratamento

Usamos os teus dados exclusivamente para:

  • Criar e gerir a tua conta (incluindo autenticação via Firebase).
  • Facilitar a conexão entre trabalhadores e empreendedores: permitir que empreendedores visualizem perfis de trabalhadores para fins de contratação.
  • Melhorar o aplicativo (análise anonimizada de uso).
  • Gerir recuperação de senha via email (gerida pelo Firebase).

Os dados não são usados para fins comerciais, marketing ou qualquer outro propósito não relacionado à facilitação de emprego em Cabo Verde.

4. Partilhamento de Dados

Os teus dados são partilhados de forma limitada:

  • Entre utilizadores do app: Empreendedores podem visualizar dados de trabalhadores para fins de contratação. Trabalhadores e empreendedores não visualizam dados uns dos outros fora desse contexto.
  • Prestadores de serviços: Com o Firebase (Google) para autenticação, armazenamento e recuperação de senha.
  • Administradores: Acesso restrito para gestão de utilizadores (ex.: resolução de problemas técnicos).
  • Autoridades: Apenas quando exigido por lei ou ordem judicial em Cabo Verde.

Não vendemos, alugamos ou partilhamos os teus dados com terceiros para fins não essenciais.

5. Transferências Internacionais

O Firebase implica transferências de dados para servidores fora de Cabo Verde (ex.: EUA). Garantimos que essas transferências cumprem os requisitos da lei cabo-verdiana, através de avaliações de adequação pela CNPD ou cláusulas contratuais padrão que asseguram um nível equivalente de proteção.

6. Conservação dos Dados

Conservamos os teus dados apenas pelo tempo necessário:

  • Enquanto a conta estiver ativa.
  • Até 5 anos após inatividade ou eliminação da conta, para fins legais ou de auditoria.
  • Dados de login no Firebase seguem as políticas do Google, mas podemos solicitar eliminação.

Após o período, os dados são anonimizados ou eliminados de forma segura.

7. Medidas de Segurança

Implementamos medidas técnicas e organizacionais adequadas, incluindo:

  • Controlo de acesso baseado em roles (trabalhadores, empreendedores, administradores).

O Firebase fornece segurança adicional para autenticação e armazenamento.

8. Os Teus Direitos

Como titular de dados em Cabo Verde, tens direito a:

  • Aceder aos teus dados.
  • Retificar dados incorretos.
  • Opor-te ao tratamento.
  • Portabilidade dos dados.

Para exercer direitos, contacta-nos em ativatrabalhosuporte@gmail.com. Responderemos no prazo de 30 dias. Se necessário, também pode apresentar reclamação junto da autoridade de proteção de dados competente.

10. Menores de Idade

O aplicativo é destinado exclusivamente a maiores de 18 anos. Não recolhemos dados de menores intencionalmente. Se detetarmos dados de menor, eliminamo-los imediatamente.

11. Atualizações desta Política

Podemos atualizar esta política para refletir mudanças no app ou na legislação. Notificaremos alterações significativas via notificação no app ou email.
